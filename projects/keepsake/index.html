Keepsake

Back to Projects

A digital love box — a personalised React app that unlocks a surprise note or memory every 8 hours.

The idea

It was 2020 and we were locked down during COVID — we couldn't see each other. I wanted to give someone a gift that kept giving. Not flowers that wilt or a card that gets forgotten in a drawer — something they could open again and again over months. The concept: a locked box that reveals one curated note every 8 hours, filled with memories, reasons I love them, inside jokes, and little quotes.

How it works

The app is built in React with Firebase on the backend. When you open the box, it pulls a random note from the database and starts an 8-hour countdown timer. Once the timer runs out, the box glows and you can open it again. Notes are categorised — memories, quotes, cute things, reasons — and you can browse past unlocks too.

There's also a hidden admin panel where I could sneak in new notes over time, so the box never ran dry.

The stack

  • React — frontend, with React Router for navigation
  • Firebase — Realtime Database, Authentication, and Cloud Storage for images
  • Semantic UI React — UI components
  • react-countdown — the all-important 8-hour timer

Was it worth it?

Yes. 10/10 would over-engineer a love letter again.
